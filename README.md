# Iso Pasianssi

Selainpohjainen pasianssipeli, jossa kaikki 52 korttia jaetaan neljään riviin. Tavoitteena on järjestää jokaiselle riville yhden maan kortit järjestykseen 2–K.

## Pelaa

Peli toimii GitHub Pagesissa:

[https://mulkmulkmulk.github.io/iso-pasianssi/](https://mulkmulkmulk.github.io/Iso-Pasianssi/)

## Säännöt

* Pelin alussa kortit jaetaan neljään 13 kortin riviin.
* Ässät poistetaan sivuun ja niiden paikoille jää tyhjät aukot.
* Vasemman reunan aukkoon saa siirtää vain 2:n.
* Muuhun aukkoon saa siirtää vain saman maan seuraavan kortin.
* Esimerkiksi hertta 8 voidaan siirtää hertta 7:n oikealle puolelle.
* Kun peli jumittuu, kortit voi jakaa uudelleen.
* Uudelleenjaossa valmiit korttiketjut jäävät paikoilleen.
* Uudelleenjaon voi tehdä kaksi kertaa.
* Peli on läpi, kun jokaisella rivillä on yhden maan kortit järjestyksessä 2–K.

## Ohjaus

Klikkaa korttia ja klikkaa sen jälkeen tyhjää paikkaa, johon haluat siirtää kortin.

## Teknologia

Peli on tehty yhdellä HTML-tiedostolla, jossa on mukana CSS ja JavaScript. Korttikuvat ovat erillisessä `svgcard`-kansiossa.
